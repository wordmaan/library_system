# 📚 Library Management System (LMS)

A full-featured Library Management System built to manage books, users, and issue/return operations efficiently.

## 🚀 Features

* 📖 Add, update, and delete books
* 👤 User management system
* 📥 Issue and return books
* 🔍 Search functionality
* 📊 Book availability tracking

## 🛠 Tech Stack

* Frontend: React.js
* Backend: flask
* Database: xammp
